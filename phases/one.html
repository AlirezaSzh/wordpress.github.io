گزارش فاز اول پروژه طراحی افزونه وردپرس

تاریخ شروع فاز اول: ۱۴۰۳/۰۹/۱۹
تاریخ پایان فاز اول: ۱۴۰۳/۰۹/۲۷
هدف فاز اول: پیاده‌سازی بخش‌های مربوط به تحلیل و طراحی، مدیریت ایونت‌ها، و مدیریت کسب‌وکارها.

جزئیات کارهای انجام‌شده در فاز اول

1. تحلیل و طراحی

  1. جمع‌آوری نیازمندی‌ها:

    • برگزاری جلسه با کارفرما جهت مشخص‌کردن تمامی جزئیات و نیازمندی‌های پروژه.
    • درک کامل اهداف افزونه و نحوه تعامل کاربران و مدیران سایت.

  2. مستندسازی و طراحی نقشه راه:

    • مستند کردن تمامی ویژگی‌های افزونه (ویژگی‌های ایونت‌ها، کسب‌وکارها، جستجو، و غیره).
    • تهیه نقشه راه پروژه به‌صورت گام‌به‌گام برای فازهای بعدی.

  3. طراحی ساختار پایگاه داده:

    • تعریف جداول و فیلدهای مورد نیاز برای ذخیره‌سازی اطلاعات ایونت‌ها و کسب‌وکارها:
      • جدول ایونت‌ها: شامل فیلدهایی مانند id, title, date, location, category, expiration_date, و غیره.
      • جدول کسب‌وکارها: شامل فیلدهایی مانند id, name, description, working_hours, contact_link, و غیره.
    • ایجاد ارتباطات بین جداول برای مدیریت داده‌ها.

  4. ساختار پوشه‌ها و فایل‌ها:

    • طراحی ساختار پوشه‌ها و فایل‌های افزونه بر اساس استانداردهای وردپرس.
    • ایجاد فایل‌های اصلی افزونه شامل:
      • فایل اصلی: britishi-plugin.php
      • پوشه includes/ برای منطق افزونه.
      • پوشه assets/ برای فایل‌های استاتیک (CSS, JS).
      • پوشه templates/ برای نمایش اطلاعات.

2. مدیریت ایونت‌ها

  1. ایجاد نوع پست سفارشی (Custom Post Type) برای ایونت‌ها:

    • ثبت یک پست تایپ سفارشی به نام event در افزونه.
    • افزودن ویژگی‌هایی مانند public, has_archive, و supports به پست تایپ.

  2. تعریف فیلدهای مرتبط با ایونت‌ها:

    • افزودن فیلدهایی مانند:
      • عنوان ایونت (Event Title): نام ایونت.
      • تاریخ ایونت (Event Date): زمان برگزاری.
      • مکان (Location): موقعیت مکانی ایونت.
      • دسته‌بندی (Category): موضوع ایونت.
      • توضیحات (Description): توضیحات مرتبط.
      • تاریخ انقضاء (Expiration Date): برای حذف خودکار ایونت‌های قدیمی.

  3. نمایش لیست ایونت‌ها در پیش‌خوان وردپرس:

    • افزودن ستونی برای نمایش تاریخ انقضاء و وضعیت فعال بودن ایونت.
    • امکان فیلتر ایونت‌ها براساس دسته‌بندی و تاریخ.

  4. قابلیت مدیریت توسط کاربران:

    • طراحی فرم ثبت ایونت توسط کاربران.
    • ارسال ایونت‌ها برای تأیید توسط مدیر سایت قبل از انتشار.

3. مدیریت کسب‌وکارها

  1. ایجاد نوع پست سفارشی برای کسب‌وکارها:

    • ثبت یک پست تایپ سفارشی به نام business در افزونه.
    • افزودن ویژگی‌هایی مشابه ایونت‌ها با فیلدهای مختص کسب‌وکار.

  2. تعریف فیلدهای مرتبط با کسب‌وکارها:

    • فیلدهای تعریف‌شده شامل:
      • نام کسب‌وکار (Business Name): نام رسمی کسب‌وکار.
      • توضیحات (Description): توضیحات مختصر.
      • ساعات کاری (Working Hours): بازه زمانی فعالیت کسب‌وکار.
      • لینک تماس (Contact Link): لینک وب‌سایت یا شماره تماس.
      • مکان جغرافیایی (Location): برای نمایش روی نقشه.

  3. نمایش لیست کسب‌وکارها در پیش‌خوان وردپرس:

    • افزودن قابلیت مرتب‌سازی و فیلتر براساس نام، دسته‌بندی و مکان.

  4. قابلیت مدیریت توسط کاربران:

    • طراحی فرم ثبت کسب‌وکار مشابه فرم ایونت.
    • ارسال کسب‌وکارها برای تأیید توسط مدیر سایت قبل از انتشار.

دستاوردهای فاز اول

  1. تحلیل و طراحی کامل:

    • نیازمندی‌ها مستندسازی و نقشه راه تهیه شد.
    • پایگاه داده و ساختار افزونه به‌طور کامل طراحی شد.

  2. مدیریت ایونت‌ها:

    • ثبت ایونت‌ها و مدیریت آن‌ها در پیش‌خوان وردپرس.
    • امکان افزودن ایونت توسط کاربران با تأیید مدیر.

  3. مدیریت کسب‌وکارها:

    • ثبت و مدیریت کسب‌وکارها در پیش‌خوان وردپرس.
    • طراحی فرم ثبت برای کاربران با قابلیت تأیید توسط مدیر.

پیشنهادات برای فازهای بعدی

  1. فیلدهای داینامیک:
    • پیاده‌سازی قابلیت افزودن فیلدهای خاص برای هر دسته‌بندی.
  2. نمایش نقشه:
    • ادغام با Google Maps یا OpenStreetMap برای نمایش ایونت‌ها و کسب‌وکارها.
  3. جستجو و فیلتر:
    • پیاده‌سازی سیستم جستجو و فیلتر پیشرفته برای کاربران.
